# E-Commerce Business Performance Analysis with SQL

How to Reproduce

To reproduce this project:

Download the Olist dataset from Kaggle.
Place the dataset archive in the project folder.
Run 01_create_sqlite_database.py to create the SQLite database.
Run 02_export_sql_outputs.py to export the SQL outputs.
Open Tableau Public and connect to the CSV files in the outputs/ folder.
Build or review the dashboard using the exported reporting tables.
